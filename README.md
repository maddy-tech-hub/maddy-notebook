# maddy-notebook

This repository is organized for two things only:

- documentation notes
- React practice examples

## Folder structure

- `docs/dotnet` contains C#, .NET, ASP.NET Core, interview, and SOLID notes
- `docs/react` contains React learning notes and interview questions
- `docs/assets` contains images used by the notes
- `src/Practice/ReactConcepts` contains the React practice examples you already created
- `src/App.tsx` renders the React practice examples in one place

## Run the React practice app

```bash
npm install
npm start
```

## Other useful commands

```bash
npm test
npm run build
```

## Notes

- this is a small practice app setup, so the heavy custom webpack config was removed
- `dist/` is generated output and should not be kept in the repo
- `node_modules/` is local dependency data and is already ignored by Git
